Product Vision

Build a premium handcrafted resin art brand website that immediately communicates elegance, craftsmanship, personalization, and trust.

This should not feel like a generic e-commerce website.

It should not feel like an AI demo.

It should not feel like a portfolio.

Instead, it should feel like visiting a luxury handcrafted boutique where every scroll creates emotional attachment toward personalized resin art.

The AI preview is only a supporting experience.

The real product being sold is:

Emotion.

Memories.

Handmade craftsmanship.

Primary Objective

Convert visitors into WhatsApp leads.

Everything on the website should move users naturally toward contacting the owner.

Not forcing.

Guiding.

Secondary Objective

Allow visitors to preview personalized resin artwork before placing an order using Your Resin Preview.

This builds confidence before purchase.

Prompt Integration

A file named:

prompt.txt

exists in the project root.

This file contains the master image-generation prompt for personalized resin previews.

The application must:

Load this prompt.
Replace its placeholders with user-provided values.
Pass the resulting prompt to the selected image generation model.
Keep the original prompt.txt unchanged so it serves as the reusable template for all generations.
